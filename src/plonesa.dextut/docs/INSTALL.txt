plonesa.dextut Installation
---------------------------

To install plonesa.dextut using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``plonesa.dextut`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        plonesa.dextut
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
